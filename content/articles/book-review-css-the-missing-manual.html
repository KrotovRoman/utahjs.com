--- title: "Book Review: CSS: the Missing Manual" author: Ken date: 2011-06-22 --- CSS: The Missing Manual is fantastic for learning and reviewing CSS. Each example builds on the previous one, illustrating not only CSS concepts but how they are used together. The book explains each CSS property one by one so it is great as a reference manual. There are plenty of screenshots for those who learn best by visual example. It discusses browser compatibility issues and browser quirks in Internet Explorer as well as ways to avoid problems. I recommend this book to all web developers.
